#### 1. 설정수정

react 버전 16이상에서 사용하기 위해서는 추가 설정이 필요

`npm i --save-dev enzyme-adapter-react-16`  

```javascript
// setupTests.js
import { configure } from 'enzyme';
import Adapter from 'enzyme-adapter-react-16';

configure({ adapter: new Adapter() });
```

```json
// package.json 추가
  "jest": {
    "setupFiles": [
      "./test/setupTests.js"
    ]
  }
```
